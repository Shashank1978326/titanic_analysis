# Titanic Dataset Analysis

This project analyzes the Titanic dataset using Python and Pandas.

## Features
- Data cleaning and handling missing values
- Feature creation (FamilySize, FarePerPerson)
- Statistical analysis
- Filtering passengers
- Report generation

## Technologies Used
- Python
- Pandas

## How to Run
1. Install pandas:
   pip install pandas

2. Run the program:
   python titanic_analysis.py

## Output
- Generates a report file: titanic_report.txt
